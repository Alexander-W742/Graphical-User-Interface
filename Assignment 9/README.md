# Name Formatter App

## Description
This application is a simple Name Formatter program created using Python and PySide6. 
The user can enter text and either capitalize or reverse the text using the GUI buttons.

## MVC Overview
- Model: Handles the text formatting logic
- View: GUI built in Qt Designer
- Controller: Connects button clicks to model functions and updates the GUI

## How to Run
Run the application in main.py
